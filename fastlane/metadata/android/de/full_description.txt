<i>Passky</i> ist ein einfacher Passwort-Manager, der auf einer Zero-Trust-Architektur arbeitet. Das bedeutet, dass nur der Benutzer seine Passwörter entschlüsseln kann. So können Benutzer ihre Passwörter sicher auf jedem Server speichern. Das heißt, wenn ein Server, auf dem alle Passwörter gespeichert sind, gehackt wird, kann der Hacker die Passwörter nicht entschlüsseln und die Daten auf diesem Server sind für ihn nutzlos.
